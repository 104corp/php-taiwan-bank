# Taiwan Bank code

[![Build Status](https://travis-ci.org/104corp/php-taiwan-bank.svg?branch=master)](https://travis-ci.org/104corp/php-taiwan-bank)

台灣的銀行代碼查詢 SDK for PHP

## 安裝

使用 Composer 安裝

```
$ composer require 104corp/taiwan-bank
```
